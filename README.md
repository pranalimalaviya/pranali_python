Project Title: Fundamental Booster – Personal Data Collector

Description:
This project is a simple Python program that collects personal information from the user and displays details about the data. It also demonstrates how Python handles different data types and memory allocation.

Features:
Takes user input such as name, age, height, and favourite number
Displays the data type of each input
Shows memory address using id() function
Calculates approximate birth year

Technologies Used:
Python 3

How to Run:
Install Python on your system
Open terminal or command prompt

Run the file using: python Fundamental_booster.py

Concepts Used:
Input and Output
Data Types (int, float, string)
Type Casting
Memory Address using id()
Basic Calculation

Limitations:
No error handling for wrong input
Birth year is approximate

Future Improvements:
Add error handling
Use current year dynamically
Store data in file
